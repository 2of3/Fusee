1. Zip the content in the template folder (e.g. FuseeConfig) and name it as the folder.
2. Copy the zip file to C:\Users\[user name]\My Documents\Visual Studio [Version]\Templates\ItemTemplates\Visual C#\FUSEE\.
3. Open Visual Studio and load the FUSEE engine.
4. You can now use the template to add a new config file via the "Add -> New Item" dialog for example.
